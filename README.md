# asyc-await-demo-node-dotnet

Demo written in .NET Core and Node.js in order to demonstrate basic asyc-await usage in both langauage.

Setup
======
- clone the repo
- Node.JS
  - Go to node directory
  - run npm i
  - run node index.js
- .NET Core
  - Go to dotnet directory
  - run dotnet run
